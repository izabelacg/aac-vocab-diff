Vocab Diff

Compare two .ce vocabulary snapshots to see exactly what changed — pages added or removed, buttons added, removed, or modified. How do I get my .ce file?

Privacy: Your files are processed on this server only — never forwarded to any third party for processing. They are saved to a temporary location on the server for the duration of the comparison and deleted immediately after. If you are accessing this via a public URL, your upload travels encrypted over the internet to the server; it is not stored anywhere along the way.

What does this compare?

It compares:

  • Pages added or removed
  • Buttons added or removed
  • Changes to: label, spoken message, pronunciation, visibility, and navigation actions

It does not compare:

  • Button position or layout (where on the grid a button sits)
  • Button images (only text fields are compared)

How buttons are identified: by label and spoken message — not position. If two buttons on the same page share the same label and message and one changes, the tool shows "one removed, one added" rather than "one modified".

Tested with TouchChat™ WordPower 60 Basic SS. Should work with other vocabulary types — if yours gives an error, please send feedback.

This server counts page visits and successful comparisons to understand usage. No personal data, filenames, or file contents are recorded.

Need to export your vocabulary? In TouchChat™: tap Vocab → Menu → Import/Export Vocab → Export and Share, then save the file to your Files app. Full instructions on TouchChat™’s support page.
